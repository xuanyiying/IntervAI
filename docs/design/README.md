# Design Documentation

This directory contains high-level business logic and design documents for the IntervAI platform.

## Documents

- **[Business Flow Analysis](./business-flow.md)** (`business-flow.md`)
  - Detailed analysis of core business processes.
  - Interaction logic for Chat, Resume Upload, and Optimization.
  - Gap analysis of current implementation vs desired state.

- **[Business Model](./business-model.md)** (`business-model.md`)
  - Commercial model and value proposition.
  - Freemium tier definitions (Free vs Pro vs Enterprise).
  - Quota management system logic.

## Structure

The design documents bridge the gap between business requirements and technical implementation. They are updated periodically to reflect the current state of the system.
