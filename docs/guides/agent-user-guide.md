# Agent User Guide

## Introduction

Welcome to the Agent system! This guide will help you get the most out of three powerful AI agents designed to help you prepare for job interviews:

1. **Pitch Perfect Agent** - Create compelling personal introductions
2. **Strategist Agent** - Build customized interview question banks
3. **Role-Play Agent** - Practice with realistic mock interviews

## Getting Started

### Prerequisites

- Active account on the platform
- Valid JWT authentication token
- Resume in digital format (PDF or text)
- Target job description

### Accessing the Agents

All agents are accessible through the web interface at `https://app.example.com/agents` or via API at `https://api.example.com/api/agents`.

## Pitch Perfect Agent

### What It Does

The Pitch Perfect Agent helps you create optimized personal introductions tailored to specific job positions. It analyzes your resume and the job description to generate compelling 30-second and 60-second pitches in different styles.

### How to Use

1. **Prepare Materials**: Gather your resume and target job description.
2. **Generate Pitch**: Upload your resume, paste the job description, select style (Technical, Managerial, Sales) and duration.
3. **Review Results**: The agent provides the generated pitch, key highlights, keyword overlap, and suggestions.
4. **Refine**: Provide feedback to adjust the pitch (e.g., "Make it more technical").

## Strategist Agent

### What It Does

The Strategist Agent analyzes your background and target role to create a personalized interview question bank. It combines common industry questions with custom questions tailored to your experience level.

### How to Use

1. **Generate Question Bank**: Upload your resume, paste the job description, and select your experience level.
2. **Review Questions**: Questions are organized by Category (Technical, Behavioral, Scenario), Priority, and Difficulty.

## Role-Play Agent

### What It Does

The Role-Play Agent simulates a realistic mock interview. It asks questions, listens to your responses, and provides real-time feedback and follow-up questions.

### How to Use

1. **Start Interview**: Set the job description, interviewer style, and focus areas.
2. **Respond to Questions**: Provide your answers naturally. The agent will analyze your response and ask follow-up questions.
3. **Conclude and Feedback**: After the interview, receive a comprehensive score, strengths, and areas for improvement.
