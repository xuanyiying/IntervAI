# ✨ 功能特性 (Features)

IntervAI 致力于为求职者提供全流程的智能化辅助，核心功能涵盖简历优化、模拟面试及面试复盘。

## 🔍 智能简历分析与优化 (Smart Resume)

基于大模型深度解析简历内容，消除与职位描述 (JD) 之间的信息差。

- **深度解析引擎**
  - 支持 PDF, DOCX, TXT 多格式解析
  - 自动提取个人信息、教育背景、工作经历、技能栈
  - 自愈式 JSON 修复逻辑，确保数据结构 100% 准确

- **JD 匹配度分析**
  - 基于向量相似度 (Vector Similarity) 计算简历与职位的匹配百分比
  - 语义分析识别简历中缺失的关键技术词汇 (Keywords Gap Analysis)
  - 提供针对性的优化建议，提升 ATS (招聘管理系统) 通过率

- **AI 内容润色**
  - **成果量化**: 自动将平铺直叙的职责描述转化为 STAR 法则 (Situation, Task, Action, Result) 格式的量化成果
  - **语言优化**: 提升专业度与表达逻辑
  - **多语言支持**: 支持中英双语简历优化

- **专业导出**
  - 内置多套符合行业标准的精美简历模板
  - 一键导出高清 PDF，保留 ATS 友好的文本层

## 🤖 沉浸式模拟面试 (AI Interview)

提供高度拟真的面试演练环境，解决"面试无准备"的痛点。

- **智能题库生成**
  - 根据简历内容和目标职位自动生成 10-15 道定制化面试题
  - 覆盖技术考察、项目深挖、行为面试 (Behavioral Questions) 等多维度

- **实时语音交互**
  - 支持语音/文字双模态交互
  - AI 面试官具备追问能力，模拟真实面试压力
  - 实时语音转文字 (STT) 与 文字转语音 (TTS)

- **面试复盘报告**
  - **多维度评分**: 从技术深度、表达逻辑、沟通技巧三个维度打分
  - **参考回答**: 为每个问题提供标准参考答案与改进建议
  - **全流程记录**: 保存完整对话录音与文字记录，形成个人成长档案

## 🛡️ 企业级特性 (Enterprise Ready)

- **商业化配额管理**
  - 基于 Redis 的高精度配额控制 (Quota Service)
  - 支持按小时 (优化次数) 和按月 (面试次数) 的混合限流策略
  
- **安全与合规**
  - 数据加密存储
  - 完整的 OAuth 认证体系 (Google, GitHub)
  - 支持本地模型部署 (Ollama) 以满足隐私需求

## 🧩 扩展性

- **插件化架构**: 核心 AI 处理流基于 LangChain LCEL 构建，易于扩展新模型
- **多模型路由**: 支持动态切换 OpenAI, Claude, DeepSeek, Qwen 等主流模型
