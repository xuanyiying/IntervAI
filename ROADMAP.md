# 🗺️ 路线图 (Roadmap)

本文档记录 IntervAI 项目的未来发展规划与待办事项。

## 🚀 近期规划 (Q1 2026)

### 核心功能增强
- [ ] **增强 Auth 模块** (优先级: Medium)
  - 集成更完善的限流策略 (Rate Limiting)
  - 增加密码强度策略验证
  - 支持双因素认证 (2FA)
- [ ] **简历编辑器增强**
  - 实现编辑器数据的云端同步与持久化 (解决目前仅本地存储的问题)
  - 打通 Chat 与 Builder 的交互链路，支持从对话一键跳转编辑

### 架构优化
- [ ] **测试覆盖率提升**
  - 核心服务 (Payment, Auth) 单元测试覆盖率提升至 70%
  - 引入 E2E 测试流程 (Cypress/Playwright)
- [ ] **可观测性完善**
  - 接入 ELK 或 Loki 日志系统
  - 完善 Prometheus 监控指标 (业务指标监控)

## 📅 中期规划 (Q2 2026)

### 业务扩展
- [ ] **多语言国际化 (i18n)**
  - 前端界面支持多语言切换
  - AI 提示词 (Prompt) 国际化适配
- [ ] **企业版功能**
  - 团队协作账号
  - 批量简历解析与人才库管理
  - 定制化面试题库导入

### 技术升级
- [ ] **微服务拆分 (按需)**
  - 评估将 AI 处理层 (Agent) 拆分为独立 Python 服务 (便于接入更丰富的 AI 生态)
- [ ] **边缘计算**
  - 利用 Edge Functions 处理简单的简历预处理任务

## ✅ 已完成 (Completed)

- [x] **服务重构**
  - Interview 模块拆分为 QuestionGenerator, Session, History 等独立服务
  - Payment 模块实现基于事务的可靠支付流程
- [x] **文档体系**
  - 建立完整的文档目录结构 (Guide, Design, Technical)
  - 规范化 API 文档与部署指南
- [x] **基础设施**
  - 统一的 Docker Compose 部署方案
  - 自动化 SSL 证书配置
